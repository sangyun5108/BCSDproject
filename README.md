# Money Diary 💰


## 왜 만들었을까?
- **가계부**를 평소에 만들어보고 싶어서
- **CRUD**를 구현해보고 싶어서
- **Redux**를 사용해 데이터를 관리하고 싶어서 
- **localStorage**를 통해 데이터를 저장하고 싶어서


## 내가 맡은 Part
- 달력 파트를 제외한 **가계부** 기능 구현


## Tech
- React
- Redux
- styled-components
- localStorage
